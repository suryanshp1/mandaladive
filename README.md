# 🌊 MandalaDive

An immersive, visually calming mandala-based experience designed to help users relax, focus, and explore patterns interactively.

## ✨ Features

* 🌀 Beautiful mandala visualizations
* 🎨 Smooth animations and transitions
* ⚡ Fast and lightweight frontend
* 🧠 Designed with psychological calmness in mind
* 📱 Responsive across devices

## ⚙️ Setup Locally

```bash
git clone https://github.com/suryanshp1/mandaladive.git
cd mandaladive

# install deps (if applicable)
npm install

# run locally
npm run dev
```

## 📦 Build

```bash
npm run build
```

## 🚀 Deployment

This project uses **GitHub Actions** for automatic deployment.

Every push to `main`:

1. Builds the project
2. Uploads artifacts
3. Deploys to GitHub Pages
